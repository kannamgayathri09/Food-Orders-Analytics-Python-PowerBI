# 🍽️ Food Orders Data Analysis Project

### Python + Power BI Integration Project

## 📌 Project Overview

This project demonstrates an end-to-end Data Analytics workflow by integrating **Python** with **Power BI**.

* **Data Cleaning:** performed using Python
* **Exploratory Data Analysis (EDA):** performed using Python
* **Business Insights:** generated using Python
* **Interactive Dashboard:** created using Power BI

---

## 🎯 Project Objective

To analyze online food delivery data and generate business insights such as:

* **Revenue Analysis by City**
* **Orders Analysis by City**
* **Payment Method Analysis**
* **Restaurant Performance Analysis**
* **Cuisine Performance Analysis**
* **Delivery Time Analysis**

---

## 🛠️ Tools & Technologies Used

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**
* **Power BI**
* **CSV Dataset**

---

## 🧹 Data Cleaning Steps (Python)

* **Handled missing values** in Discount_Applied using Mode
* **Filled missing values** in Discount_Amount using Mean
* **Checked duplicate records**
* **Performed Outlier Detection** using IQR Method
* **Prepared a clean dataset** for analysis

---

## 📊 Exploratory Data Analysis (EDA)

### **1️⃣ Revenue Analysis by City**

Used `groupby()` to calculate total revenue generated by each city.

### **2️⃣ Orders Analysis by City**

Calculated total number of orders placed in each city.

### **3️⃣ Payment Method Analysis**

Analyzed order distribution across payment methods.

### **4️⃣ Restaurant Rating Analysis**

Calculated average customer ratings for each restaurant.

### **5️⃣ Cuisine Rating Analysis**

Compared average customer ratings across cuisine types.

### **6️⃣ Delivery Time Analysis**

Analyzed average delivery time across different cities.

## 🔍 Key Insights

* **Chennai generated the highest revenue**
* **Wallet was the most preferred payment method**
* **Chennai recorded the highest number of orders**
* **Biryani House received the highest average customer rating**
* **Japanese cuisine achieved the highest average customer rating**
* **Hyderabad experienced the longest average delivery times**

**Python** was used for data cleaning, preprocessing, and exploratory data analysis, while **Power BI** was used to build an interactive dashboard for business reporting.
